# PRUMアカデミー

## ホームケアナビ作成用PJ

### 環境構築
 - （初回）docker image build && db create

```
make create
```

- docker run

```
make run
```

#### 以下環境が立ち上がっていればOK

 - http://localhost:3000
 - http://localhost:80

password | pg-password
